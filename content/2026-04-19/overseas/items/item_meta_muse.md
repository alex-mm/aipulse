- **[待证实]** Meta 成立 **Meta Superintelligence Labs（MSL）**，发布首款专有模型 Muse Spark

### 关键细节

**[待证实] Muse Spark 核心特性：**
- **原生多模态**：文本/图像/视频/空间上下文一体化处理
- **Contemplating Mode**：多 subagent 并行推理，类似 Gemini Deep Think / Claude Extended Thinking
- **"people-first"定位**：强调以人为中心的 AI 体验
- **不再开源**：与 Llama 系列的开源路线彻底分道扬镳

### 解读：这意味着什么

> 以下为编辑分析，非原文内容

- **解读**：如果属实，Meta 放弃 Llama 开源路线将是 AI 行业的重大战略转向。Meta 此前以"开源对抗 OpenAI 闭源"为核心战略，Llama 系列是这一战略的旗帜。转向闭源意味着 Meta 认为开源路线无法在超级智能竞争中获胜
- **解读**：Contemplating Mode（多 subagent 并行推理）与 Claude Opus 4.7 的 Long Horizon Task、OpenAI Agents SDK 的多 Agent 编排方向一致，说明"多 Agent 并行推理"正在成为顶级模型的标配能力
- **解读**：如果 Meta 真的放弃开源，开源大模型生态将面临重大空缺——Llama 是目前最广泛使用的开源基础模型之一。这个空缺可能由 Qwen（阿里）、Mistral（法国）或其他开源模型填补
- **解读**：在 Meta 官方确认之前，不建议将 Muse Spark 纳入产品规划或竞争分析。关注 Meta 官方博客（blog.meta.com）和 Connect/F8 等活动的公告
